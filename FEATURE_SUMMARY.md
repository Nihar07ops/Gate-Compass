# GateCompass - Feature Summary

## Current Features

### Mock Tests
- **GATE Format**: 65 questions matching actual GATE pattern
- **Custom Format**: 20 questions for quick practice
- **Difficulty Levels**: Easy, Medium, Hard
- **Test Modes**: Timed (with countdown) & Freestyle (no limit)

### Question Bank
- **59 GATE Professional Questions**
- **Quality**: No definitions, application-focused
- **Types**: 71% numerical, 29% complex MCQ
- **Topics**: OS, Algorithms, DBMS, Networks, TOC, COA, Digital Logic, Aptitude

### User Features
- Secure authentication (JWT)
- Test history tracking
- Performance analytics
- Progress monitoring
- Streak counter

### Analytics
- Topic-wise performance
- Score trends
- Strength/weakness analysis
- Detailed statistics
- Visual charts

### ML-Powered
- Predictive analysis
- Topic importance scoring
- Personalized recommendations
- Performance forecasting

### Additional
- Historical trends (GATE patterns)
- Study resources
- Responsive design
- Clean, professional UI

## Technical Stack

**Frontend**: React.js, Material-UI, Framer Motion, Vite  
**Backend**: Node.js, Express, JWT, MongoDB/In-memory  
**ML Service**: Python, Statistical analysis
